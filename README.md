# Dev Container Features: andy-todd-dev

A collection of [dev container Features](https://containers.dev/implementors/features/) published to GitHub Container Registry (GHCR) and following the [dev container Feature distribution specification](https://containers.dev/implementors/features-distribution/).

## Features

### [`dart`](src/dart/README.md)

Installs the Dart SDK with configurable version and release channel, with the Dart VS Code extension pre-configured.

### [`android-sdk`](src/android-sdk/README.md)

Installs the Android SDK with configurable platform versions, build tools, NDK, and CMake support.

### [`flutter`](src/flutter/README.md)

Installs a Flutter release (and its bundled Dart SDK) into the container, with Dart and Flutter VS Code extensions pre-configured.
